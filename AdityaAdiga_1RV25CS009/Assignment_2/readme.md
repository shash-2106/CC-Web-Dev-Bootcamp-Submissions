# My To-Do List App

## What I built
A simple To-Do List app using HTML, CSS and JavaScript.

You can add tasks and delete them when done.

## Features
- Type a task and click "Add Task" to add it to the list
- Each task has a Delete button to remove it
- Press Enter to add a task quickly
- About Me section with my name and avatar

## How to run it
1. Download all the files
2. Open `index.html` in your browser
3. That's it!

## Files
- `index.html` - the main page
- `style.css` - all the styling
- `script.js` - the JavaScript logic
- `README.md` - this file
