# Assignment 1: Simple To-Do App

## Overview
A modern, responsive, and minimalist web-based To-Do List application built as part of the CC Web Dev Bootcamp Submission. It allows users to easily keep track of their daily tasks.

## Features
- **Dynamic Task Management:** Add, delete, and check off tasks.
- **Input Validation:** Prevents adding empty or whitespace-only tasks with visual feedback.
- **Task the Checkboxes:** Click the custom-styled checkbox to marked tasks as completed (strikes through text).
- **Clear All Functionality:** Quickly remove all items from the list. The "Clear All" button auto-disables when the list is empty to prevent unnecessary clicks.
- **Premium UI/UX:** Clean, centered card layout featuring custom font (Inter), smooth micro-animations, glassmorphism hints, and tailored color palette (dark mode aesthetic).
- **Pure Frontend:** Built entirely using vanilla HTML, CSS, and Javascript. No backend or external libraries (other than Google Fonts) required.

## Structure
- `index.html`: Contains the core structure and layout.
- `style.css`: Contains all custom styling and animations.
- `script.js`: Handles logic for manipulating the DOM dynamically.
